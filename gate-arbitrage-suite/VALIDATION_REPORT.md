## 📦 Installation Instructions

### Quick Start:
```bash
# 1. Install Hummingbot
git clone https://github.com/hummingbot/hummingbot.git
cd hummingbot
./install.sh

# 2. Copy arbitrage suite
cp -r gate-arbitrage-suite/controllers/* hummingbot/smart_components/controllers/
cp -r gate-arbitrage-suite/scripts/* scripts/
cp -r gate-arbitrage-suite/conf/* conf/

# 3. Start Hummingbot
./start.sh

# 4. Run arbitrage
>>> connect gate_io
>>> connect gate_io_perpetual
>>> start --script gate_arb_v2.py
```

### Web UI (Optional):
```bash
cd gate-arbitrage-suite/webui
./install.sh
./start.sh
# Access at http://localhost:8080
```
